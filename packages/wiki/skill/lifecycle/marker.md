# Marker

`marker` 是 handle 的 runtime projection。

当 active skills 共享 user-visible governance expectation 时，markers MAY 被 skill family 共享。

dispatcher 不 owns marker。

active skills 投影自己的 marker：`💬` 用于 expression family，`🔗` 用于 link family，`🧭` 用于 orientation family，`🎼` 用于 Partita-managed base layer，`🧹` 用于 maintenance family。

参见 [[projection/runtime]]。
