# Marker

loaded-skill marker 是 handle 面向 Codex 的可见投影。

Partita markers 属于 active skills 或 skill families，不属于 `harness/skills/dispatcher.md`。

`💬` 是 expression-family marker。

`🔗` 是 link-family marker。

`🧭` 是 orientation-family marker。

`🎼` 是 Partita-managed base-layer marker。

`🧹` 是 maintenance-family marker。

marker MUST 只在 owning skill active 时出现。

参见 [[skill/lifecycle/marker]]。
